# 進捗ログ

## 運用ルール

- 作業した日ごとに記録する
- 1日1ブロックを基本とする
- 作業時間は20分程度を目安にする
- 今日やったこと、完了したこと、次回やることを残す
- 課題が発生した場合は issue_log.md にも記録する
