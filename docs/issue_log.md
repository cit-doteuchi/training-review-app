# 課題管理ログ

## 運用ルール

- 作業中に困ったことを記録する
- 原因、対応、結果をセットで残す
- 解決済みかどうかを状態で管理する
- 小さな問題でも、学びになったものは記録する

## 状態の定義

| 状態 | 意味 |
|---|---|
| 未対応 | まだ対応していない |
| 対応中 | 現在対応している |
| 解決済み | 対応が完了した |
| 保留 | 今回は対応しない |
