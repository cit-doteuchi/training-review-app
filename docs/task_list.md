# タスク一覧

## 状態の定義

| 状態 | 意味 |
|---|---|
| 未着手 | まだ作業していない |
| 作業中 | 現在取り組んでいる |
| 完了 | 完了条件を満たした |
| 保留 | 今回は一旦後回しにする |

## 優先度の定義

| 優先度 | 意味 |
|---|---|
| 高 | MVPに必要な機能 |
| 中 | 提出物として完成度を上げる機能 |
| 低 | 余裕があれば対応する機能 |
